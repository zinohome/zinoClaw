# Additional Aesthetic Recipes (8 Specialized)

## 6. ACM Conference (acmart)

Best for: ACM proceedings (SIGCHI, SIGMOD, etc.)

| Property | Value |
|----------|-------|
| Body font | Times New Roman 9pt |
| First-line indent | ~10pt |
| Title | 14.4pt bold, left-aligned |
| Heading 1 | 10pt bold, ALL CAPS, left-aligned |
| Heading 2 | 10pt bold, left-aligned |
| Heading 3 | 10pt bold italic, left-aligned |
| Line spacing | 1.0x |
| Page margins | Top/Bottom 1.25in, Left/Right 0.75in |
| Columns | 2, gutter 0.33in |
| Captions | 8pt |
| References | 7.5pt |

## 7. APA 7th Edition (Professional Paper)

Best for: Psychology, social sciences, APA-mandated submissions.

| Property | Value |
|----------|-------|
| Body font | Times New Roman 12pt |
| First-line indent | 0.5 inch |
| Level 1 (H1) | 12pt bold, centered |
| Level 2 (H2) | 12pt bold, left-aligned |
| Level 3 (H3) | 12pt bold italic, left-aligned |
| Level 4 (H4) | 12pt bold, indented 0.5in, run-in with period |
| Level 5 (H5) | 12pt bold italic, indented 0.5in, run-in |
| Line spacing | 2.0x (double) |
| Page margins | 1in all sides |
| Header | Running head flush left ALL CAPS + page number flush right |

## 8. MLA 9th Edition

Best for: Literary studies, humanities.

| Property | Value |
|----------|-------|
| Body font | Times New Roman 12pt |
| First-line indent | 0.5 inch |
| Title | 12pt centered, NOT bold/italic (plain) |
| Line spacing | 2.0x (double) |
| Page margins | 1in all sides |
| Header | "LastName PageNumber" flush right |
| First page | Name, instructor, course, date upper-left |
| Works Cited | 0.5in hanging indent |
| Special | No heading hierarchy prescribed; radical plainness |

| Heading 1 | 12pt bold, centered |

| Heading 3 | 12pt bold, left-aligned |
| Heading 4 | 12pt NOT bold, left-aligned |

| Page margins | Top/Bottom/Right 1in, Left 1.5in (binding) |
| Footnotes | 10pt, single-spaced within |
| Block quotes | Indented 0.5in from left, single-spaced |
| Page numbers | Centered bottom |
